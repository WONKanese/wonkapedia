Chicken Basement

Mac Download: Mac

FOR MAC IF PROGRAM SAYS CANNOT BE OPENED OR SOMETHING LIKE THAT:

  1. rename the .app file to ChickenBasement
  2. Open Terminal
  3. in terminal go into the folder that contains the .app file. (For example if the file is in the downloads then do cd downloads)
  4. then do cd ChickenBasement (will be different if you named your .app file to something other than ChickenBasement)
  5. then do cd Contents/MacOS
  6. then do chmod +x ChickenBasement
  7. go back into finder and do show package contents of the .app file
  8. go into contents folder and then MacOS folder
  9. open the Chicken Basement file
