FOR MAC IF PROGRAM SAYS CANNOT BE OPENED OR SOMETHING LIKE THAT:

  • rename the .app file to ChickenBasement
  • Open Terminal
  • in terminal go into the folder that contains the .app file. (For example if the file is in the downloads then do cd downloads)
  • then do cd ChickenBasement (will be different if you named your .app file to something other than ChickenBasement)
  • then do cd Contents/MacOS
  • then do chmod +x ChickenBasement
  • go back into finder and do show package contents of the .app file
  • go into contents folder and then MacOS folder
  • open the Chicken Basement file
